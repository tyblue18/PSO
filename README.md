# PSO-Based Brain Tumor Segmentation

A medical image segmentation project using Particle Swarm Optimization (PSO) for automated brain tumor detection on MRI scans.

## Overview

This project implements a PSO-based approach to segment brain tumors from medical images, using the BraTS dataset as a benchmark.

## Requirements

- Python 3.7+
- NumPy
- OpenCV
- PySwarms
- Matplotlib
